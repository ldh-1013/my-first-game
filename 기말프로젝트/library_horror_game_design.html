01 — CONCEPT

게임 컨셉 및 핵심 철학

『폐도서관 — 새벽 1시』는 지식의 공간이었던 도서관이 망각과 공포로 변모한다는 역설을 핵심 테마로 삼습니다. 책 한 권에 담겼던 누군가의 기억이 글리치(Glitch)로 변질되며 현실을 침식하는 연출을 통해, 단순한 귀신 공포를 넘어 정보 자체가 붕괴되는 인식론적 공포를 구현합니다.

핵심 디자인 필로소피 텍스트는 신뢰할 수 없다. 플레이어가 발견하는 단서, 노트, 표지판은 언제든 글리치되어 잘못된 정보를 줄 수 있다. 공포는 어둠이 아니라 불완전한 정보에서 온다.
👁

시야 제한 + 라이터

기본 시야는 극도로 좁고, 라이터만이 주변을 밝힌다. 기름 잔량 관리가 생존의 핵심.

📖

글리치 내러티브

중요한 스토리 텍스트일수록 더 심하게 깨진다. 공포가 고조될수록 UI 자체도 붕괴.

🕐

새벽 1시→5시 타임라인

실시간 시계가 진행되며 특정 시간에 이벤트가 트리거. 시간 압박이 긴장감을 상승.

🔴

아오오니 DNA 계승

탑다운 2.5D 시점, 조우 시 즉각적 추격, 퍼즐 중심 진행, 공포의 원천은 예측 불가능한 유령 AI 패턴.

🎭

분위기 > 액션

전투 불가. 숨기와 도주만 가능. 유령을 절대 이길 수 없다는 무력감이 공포의 본질.

02 — NARRATIVE

스토리 & 세계관

배경

2031년. 10년 전 폐관된 「청운시립도서관」은 도시 재개발 논쟁의 중심에 서 있다. 주인공 김하준(22세, 대학원생)은 지역 문화재 보존 리포트를 위해 폐도서관 내부 자료를 수집하러 건물에 몰래 들어갔다가, 비상구가 봉쇄된 것을 뒤늦게 알게 된다.

그 순간, 도서관에 불이 꺼진다. 새벽 1시. 하준은 라이터 하나만을 든 채 혼자가 아니라는 것을 곧 깨닫는다.

숨겨진 설정 — 플레이어가 추리해야 하는 진실 이 도서관의 사서 「이정숙(故)」은 10년 전 도서관 화재 사고로 사망했다. 공식 기록과 달리, 그녀는 방화를 막으려다 유령이 된 것이 아니라 — 방화범 본인이었다. 도서관 곳곳에 흩어진 노트들은 그녀의 고백이며, 글리치는 그녀의 기억이 충돌하는 증거다.

글리치 레벨 시스템

글리치의 강도는 GLITCH LEVEL 1~5로 구분되며, 스토리가 진행될수록, 또는 유령과 가까워질수록 레벨이 상승합니다. 레벨 5에서는 UI 전체가 붕괴하는 연출이 발생합니다.

LEVEL 트리거 조건 시각 효과 텍스트 효과
LV.1 게임 시작 직후, 첫 노트 발견 미세한 스캔라인 떨림 일부 글자 깜빡임
LV.2 자료실 진입, 사서 유령 첫 조우 RGB 채널 분리 (Chromatic Aberration) 문장 중간 문자 깨짐
LV.3 새벽 3시, 지도 조각 수집 시작 수평 노이즈 밴드 출현 단락 전체 왜곡, 바이너리 코드 삽입
LV.4 미디어룸 진입, 진실 노트 발견 화면 세로축 시프트, 잔상 원문이 사라지고 다른 문장으로 대체
LV.5 이정숙과 최종 조우, 탈출 직전 전체 화면 분열, UI 위치 이탈 텍스트가 화면 밖으로 흘러넘침
핵심 글리치 퍼즐 — 반전 장치 자료실의 핵심 노트는 처음 읽을 때 「학습 공간 열쇠는 3번 책장 위에 있다」라고 표시된다. 하지만 글리치 레벨이 올라가 재방문하면 텍스트가 붕괴되며 실제 위치인 「지하 1층 보일러실」이 드러난다. 플레이어는 글리치를 적이 아닌 단서로 활용해야 한다.

기술 구현 — Pygame 글리치

# 글리치 효과 핵심 구현
# pygame.Surface를 슬라이스하여 수평 이동
def apply_glitch(surface, intensity):
    num_strips = random.randint(2, 8 * intensity)
    for _ in range(num_strips):
        y = random.randint(0, surface.get_height())
        h = random.randint(1, 6)
        offset = random.randint(-20*intensity, 20*intensity)
        strip = surface.subsurface(0, y, W, h).copy()
        surface.blit(strip, (offset, y))

# RGB 채널 분리 (Chromatic Aberration)
    r_surf = surface.copy()
    r_surf.fill((0,255,255), special_flags=pygame.BLEND_MULT)
    surface.blit(r_surf, (-3*intensity, 0), special_flags=pygame.BLEND_ADD)

07 — GHOST DESIGN

유령 캐릭터 설계

각 유령은 도서관에 얽힌 사연이 있으며, 행동 패턴이 다릅니다. 아오오니처럼 추격 시 별도의 추격 BGM과 가속이 적용되며, 유령마다 「무력화 조건」이 존재합니다.

👻

떠도는 책 (The Floating Book)

TYPE: PATROL / STAGE 1 전용

도서관의 원한을 품은 책들이 자율 이동합니다. 직접적인 공격은 없지만, 접근 시 날카로운 소리를 내어 다른 유령을 소환합니다. 무력화 조건: 책장에 돌려놓기 (E 버튼 상호작용).

🤍

독자 유령 (The Reader)

TYPE: STATIONARY AMBUSH / 의자에 앉아 있음

열람실 의자에 앉아 있는 유령. 라이터 빛이 닿으면 머리를 돌려 응시합니다. 5초 이상 시선이 유지되면 추격 시작. 무력화 조건: 라이터를 끄고 3초 대기.

🕯

이정숙 (The Librarian)

TYPE: PURSUIT / 메인 보스

게임 전체를 관통하는 핵심 유령. 특정 구역에 출현하여 추격합니다. 빛에도 두려워하지 않으며, 숨기도 제한적으로만 통합니다. 글리치가 심화될수록 속도가 빨라집니다. 무력화 조건: 없음. 도주만 가능.

🌫

안개 아이 (The Fog Child)

TYPE: MIMIC / 새벽 3시 이후 등장

어둠 속에서 어린 목소리로 「도와줘」라고 속삭입니다. 접근하면 순간 이동 후 공격. 라이터가 꺼진 상태에서만 활성화됩니다. 무력화 조건: 라이터를 켜는 것만으로 퇴각.

📰

기록 수집가 (The Archivist)

TYPE: THIEF / 자료실 전용

자료실에서만 등장하는 유령. 플레이어의 소지품을 빼앗아 달아납니다. 잡히면 아이템 1개 소실. 무력화 조건: 라이터로 정면 조준 후 2초 유지 (빛에 약함).

GLITCH와 유령의 상관관계 이정숙이 가까워질수록 글리치 레벨이 강제로 올라갑니다. 역설적으로 글리치된 텍스트에서 진실이 드러나므로, 플레이어는 위험을 감수하고 이정숙과 가까이 있어야 더 많은 단서를 얻을 수 있습니다.

텍스트 시스템

JSON 기반 대화 데이터. 글리치 레벨에 따라 렌더 함수가 문자를 오염. 타자기 효과 + 글리치 오버레이 혼합.

사운드

pygame.mixer 멀티채널. 발소리 거리 감쇠, 환경음 (바람, 책 넘기는 소리), 추격 BGM 페이드인.

💾

세이브 시스템

JSON 파일로 진행 상태 저장. 특정 이벤트 발생 후 자동 저장. 아오오니와 달리 세이브 포인트 3개 제공.

엔딩 A — 탈출

새벽 5시 이전에 탈출 열쇠를 획득하고 정문으로 나간다. 이정숙의 진실을 밝히지 못한 채 도망친 하준. 해피 엔딩이지만 뭔가 찜찜한 여운.

🕯

엔딩 B — 진실 엔딩

이정숙의 노트를 모두 수집하고 그녀에게 직접 말을 건다. 그녀는 서서히 사라지며 도서관의 문이 열린다. 진정한 탈출. 크레딧에 이정숙의 사연이 공개.

💀

엔딩 C — 감금

새벽 5시 이후까지 탈출 못하거나, 이정숙에게 3번 잡히면 하준은 도서관의 새로운 유령이 된다. 다음 플레이에서 하준 유령이 배경에 등장하는 이스터에그.

조명 연출 세부

라이터 불꽃 상태

기름 100~21%: 안정적 원형 시야 (반경 180px)
기름 20%~11%: 불꽃 흔들림 애니메이션, 반경 120px로 축소
기름 10%~1%: 급격한 깜빡임, 반경 60px
기름 0%: 완전 소등, 시야 0px

환경 조명

비상구 표지판: 초록 약광 (반경 30px)
일부 창문: 달빛 (파랑 약광, 반경 50px)
공중전화: 붉은 약광 이벤트 포인트
