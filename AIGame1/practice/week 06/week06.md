## 사용한 AI 프롬프트 (요약)
1. step 1,2에 있는 AI활용 과제의 프롬포트를 사용하여 코드를 작동시켜봤다.
2. 아래 코드에서 플레이어 사각형을 walk_frames로 교체하고 애니메이션이 재생되게 해줘.
3. 아래 코드에서 boom.wav를 Base64로 임베딩해줘. io.BytesIO를 사용해서 파일 없이 실행되게 해줘.
4. 게임 시작 시 bgm.ogg 무한 반복, 충돌시 hit.wav 재생하는 코드 추가해줘.
5. Sprite Sheet 이미지를 적용하는 방법을 알려줘.
6. 코드 실행 중 발생하는 다양한 오류에 대해서 의미와 해결 방법을 자세히 알려줘.


## AI 답변에서 도움이 된 것
코드 실행 중 발생하는 다양한 오류애 대해서 해당 오류는 어떤 것을 뜻하는지, 어떻게 오류의 종류를 파악할 수 있는지, 수정하는 방법 등의 답변 내용이 도움이 됐다.
## AI 답변을 수정하거나 버린 것
Sprite Sheet 이미지를 로드하지 못해 직접 코드를 추가해 로드 가능하게 만들었다.
## 적용 결과
- 잘 된 것: 게임 기획서와 조금은 유사한 게임을 만들 수 있었다.
- 어려웠던 것: Sprite Sheet의 순서를 상황에 맞게 사용할 수 있도록 코딩하는 것, 게임을 본인의 의지대로 만드는 것 등이 매우 어려웠다.
- 다음에 시도할 것: Snake Game과 관련된 Sprite Sheet를 찾기 어려워 직접 이미지를 만들어 자연스럽게 만들고, 이를 게임에 적용해 더욱 완성도 높은 게임을 만듥고자 한다.
