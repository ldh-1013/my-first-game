Snake Diet: 최종 통합 기획 사양서 (Final Version)
1. 프로젝트 개요
게임명: Snake Diet

장르: 아케이드 / 피지컬 액션

기반: 스네이크(Snake) 게임 변형

핵심 컨셉: '성장'이 아닌 **'다이어트(수축)'**를 통해 생존하고 탈출하는 역발상 게임.

비주얼 톤: 360도 자유 이동의 부드러움과 화려한 점수 팝업(Floating Text)이 강조된 스타일.

2. 게임 상태 및 승패 조건
2.1. 승리 (Clear)
점수 달성: 누적 점수 600점 달성 시 맵 무작위 위치에 탈출구 생성.

파이널 페이즈(Final Phase): 탈출구 생성 즉시 맵의 모든 방해물(햄버거, 치킨)이 플레이어를 향해 아주 천천히 다가오는 유도 성능 활성화.

최종 승리: Snake의 머리가 탈출구에 충돌 시 화면 3(Success)으로 전환.

2.2. 패배 (Game Over)
물리적 충돌: 머리가 자신의 몸통, 맵 외곽 벽, 또는 장애물(폭탄 등)에 부딪힐 때.

과부하(Overload): 뱀의 몸길이가 70칸을 초과할 때 즉시 사망 처리 후 화면 4(Failure)로 전환.

3. 조작 및 물리 엔진 (Advanced Physics)
자유 조향: 격자(Grid) 없는 360도 전 방향 자유 이동.

A / D 키: 좌/우 회전 (딜레이 0, 입력 즉시 반응).

길이-속도 역상관: 몸이 짧을수록 속도 가속, 길수록 속도 감속.

순간 수축 (Space Bar): 2.5초간 반투명(Ghost) 상태 전환.

효과: 자신의 몸통 및 장애물 통과 가능 (벽 제외), 이동 경로에 잔상 이펙트 발생.

패널티: 사용 즉시 요요 게이지 40% 상승.

시스템 키: Q(시작), R(재시작), Esc(종료).

4. 핵심 시스템
4.1. 요요 시스템 (Yoyo System)
자동 누적: 요요 게이지가 우측 상단에서 실시간으로 자동 충전.

강제 팽창: 게이지 100% 도달 시 몸마디 +1칸 강제 증가 및 게이지 초기화.

위기 연출: 게이지 90% 이상 시 화면 테두리가 빨간색으로 점멸하며 긴박함 조성.

4.2. 콤보 다이어트 (Combo System)
발동 조건: 소화제 획득 후 3초 이내에 추가 소화제 획득 시 콤보 카운트 시작.

효과: 콤보 유지 시 점수 획득량 배율 상승 및 요요 게이지 일시 정지.

5. 아이템 및 기믹 데이터 (Entities)
모든 아이템 획득 시 해당 위치에 점수 텍스트가 떠오릅니다. (+는 노란색, -는 빨간색)

일반 소화제 (흰색)

효과: 점수 +1 (노란색) / 몸 길이 -1칸

특징: 획득 즉시 맵에서 플레이어와 가장 먼 위치에 랜덤 리스폰.

고급 소화제 (금색)

효과: 점수 +5 (노란색) / 몸 길이 -3칸

특징: 획득 즉시 맵에서 플레이어와 가장 먼 위치에 랜덤 리스폰.

강력 소화제 (보라색)

효과: 점수 +30 (보라색) / 몸 길이 -30칸

특징: 플레이 중 최대 7회 출현, 고득점 수단.

햄버거

효과: 점수 -100 (빨간색) / 몸 길이 +1칸

특징: 맵 곳곳에 배치된 기본 방해 아이템.

치킨

효과: 점수 -1000 (빨간색) / 몸 길이 +3칸

특징: 누적 점수 3,000점 이상부터 등장하며, 탈출을 방해하는 고위험 요소.

의문의 알약 (무지개색)

효과: 10초간 [두께 변화(두껍게/얇게), 속도 변화(가속/감속)] 중 1~2개 중첩 발생.

특징: 전략적 선택 시스템. 1초마다 색상이 변하며 플레이어가 획득하는 시점의 색상에 따라 효과 결정.

6. 화면 구성 (Scene Flow)
화면 1: 타이틀 (Title)
UI: 좌측에 조작법 안내(A,D / Space / R / Esc), 우측에 [ Q - 게임 시작하기 ] 배치.

화면 2: 인게임 (In-Game)
UI: 좌측 상단 Score: n, 우측 상단 Yoyo Gauge: [Bar] 표시.

연출: 아이템 획득 지점에 노란색/빨간색 점수 팝업, 콤보 수치 UI 실시간 반영.

화면 3: 클리어 (Success)
중앙: "DIET COMPLETE" (노란색 굵은 텍스트).

하단: [ R - 다시 플레이 ] 문구 출력.

화면 4: 게임 오버 (Failure)
중앙: "Game Over" 텍스트 및 그 아래 "Final Score: n" 표시.

하단: [ R - Play Again ] 문구 출력.
